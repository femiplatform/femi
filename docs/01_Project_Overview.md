# FEMI Starter / Final Package

แพ็กเกจนี้เป็น “ระบบเว็บแอป FEMI” แบบเริ่มต้น (Starter) ที่ใช้งานได้จริง โดยใช้

- **Backend:** Google Apps Script (Web App) + Google Sheets (Database)
- **Frontend:** Static Web (HTML/CSS/JS) รองรับการโฮสต์ฟรี เช่น Cloudflare Pages / GitHub Pages

โครงสร้าง
- `starter/backend/` : โค้ด Apps Script (.gs) + `appsscript.json`
- `starter/frontend/` : หน้าเว็บทั้งหมด (User + Admin)
- `docs/` : เอกสารประกอบการติดตั้ง/ใช้งาน
